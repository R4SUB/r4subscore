# r4subscore: Submission Confidence Index Engine for Clinical Submission Readiness

Converts standardized R4SUB evidence into indicator scores, pillar
scores, and a Submission Confidence Index (SCI). Provides sensitivity
analysis, explainability tables, and decision band classification to
answer the question: are we ready for regulatory submission?
